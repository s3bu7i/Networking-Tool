İlk öncə Network Tool faylının içərisinə daxil oluruq.Netwok Tool faylı içərisində cmd açırıq və 
"py network.py" bunu daxil edirik. Qarşımıza cmd-də loading yazısı gələcək 100% olduqda program aktiv olacaq.
Açılan interface-də 3 seçimdən birini seçirsiniz, məsələn birincini seçirik.
Sonra cmd(command promete) də həmin tool aktiv olacaq və sizdən isdədiyi məlumatları daxil edərək verdiyi nəticələri görə bilərsiniz.
cmd-də none yazısı gədikdə program bitmiş olur və siz interface də başqa bir seçim edərək onu edə bilirsiniz.
